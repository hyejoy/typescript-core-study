# 타입 단언 (Type Assertion)

## 개념

특정 데이터가 어떠한 타입을 가진다고 확신할 때 사용하는 문법
TypeScript 컴파일러에게 "이 데이터는 내가 지정한 타입이 맞아" 라고 알려주는 역할

## 사용법

### 1. 꺽쇠 방법

```typescript
const fixedNum = (<number>num).toFixed(2);
```

### 2. as 키워드 방법 (권장)

```typescript
const fixedNum = (num as number).toFixed(2);
```

JSX에서는 꺽쇠를 인식할 수 없어서 as 키워드를 더 선호

## 타입가드 vs 타입단언

타입가드 → 조건문으로 타입을 좁혀줌 (안전)
타입단언 → 컴파일러에게 타입을 강제로 주입 (확신할 때만 사용)

## ⚠️ 주의사항

타입 단언은 컴파일러가 타입을 강제로 믿게 만들기 때문에
실제 타입이 다를 경우 컴파일 에러는 없지만 **런타임 에러 발생 가능**

```typescript
const num: unknown = "10.12"; // 실제는 문자열
const fixedNum = (num as number).toFixed(2); // 컴파일 에러 없음
// → 실행 시 에러 발생!
```

## 정리

타입 단언은 개발자가 특정 데이터의 타입을 **100% 확신할 수 있을 때만** 사용
