# 타입가드 확장

## 핵심

타입가드는 typeof 키워드가 필수가 아님
→ **타입을 좁혀줄 수 있기만 하면** 모두 타입가드

## typeof 외 타입가드 방법들

### 1. Truthy 체크

```typescript
function toUpper(a: string | null) {
  if (a) return a.toUpperCase(); // null은 false로 평가 → 타입 좁혀짐
}
```

### 2. 불일치 연산자

```typescript
function toUpper(a: string | null) {
  if (a !== null) return a.toUpperCase(); // null이 아님을 보장
}
```

### 3. 옵셔널 체이닝

```typescript
function toUpper(a: string | null) {
  return a?.toUpperCase(); // 값이 있을 때만 메서드 호출
}
```

### 4. in 키워드 (객체 속성 확인)

```typescript
function getName(obj: object) {
  if ("name" in obj) return obj.name; // obj에 name 속성이 있을 때만 참
}
```

## 정리

typeof / Truthy 체크 / 불일치 연산자 / 옵셔널 체이닝 / in 키워드 등
→ 타입을 좁혀줄 수 있다면 모두 타입가드
