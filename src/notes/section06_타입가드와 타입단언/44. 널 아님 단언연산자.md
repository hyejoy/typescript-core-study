# 널 아님 단언 연산자 (Non-null Assertion Operator)

## 개념

TypeScript에서 null 값을 처리하기 위한 특별한 연산자
`!` 기호를 사용하여 컴파일러에게 해당 데이터가 null이 아님을 단언

## 사용법

```typescript
function toFixed(num: number | null) {
  return num!.toFixed(2); // null이 아님을 단언
}
```

## null 처리 방법 비교

```typescript
// 1. Truthy 체크
if (num) num.toFixed(2);

// 2. 불일치 연산자
if (num !== null) num.toFixed(2);

// 3. 옵셔널 체이닝
num?.toFixed(2);

// 4. 널 아님 단언 연산자
num!.toFixed(2);
```

## ⚠️ 주의사항

실제 데이터가 null인 경우 컴파일 에러는 없지만 **런타임 에러 발생**

```typescript
toFixed(null); // 컴파일 에러 없음 → 실행 시 에러 발생!
```

## 정리

널 아님 단언 연산자는 개발자가 해당 데이터가 null이 아님을 **100% 확신할 수 있을 때만** 사용
