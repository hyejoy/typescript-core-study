# 타입가드 (Type Guard)

## 개념

변수의 타입을 좁혀서 TypeScript 컴파일러가 변수의 실제 타입을 추론할 수 있도록 도와주는 메커니즘
거창한 문법이 아니라 **조건문을 사용해서 타입을 좁혀주는 행위**

## 타입가드가 필요한 이유

유니온 타입 사용 시 매개변수 타입의 경우의 수가 증가
→ TypeScript는 같은 자료형의 연산만 허용하기 때문에 타입 에러 발생
→ JavaScript는 암묵적 형변환으로 허용하지만 TypeScript는 불가

## 예시

```typescript
function add(a: number | string, b: number | string): number | string {
  if (typeof a === "number" && typeof b === "number") {
    return a + b; // 숫자 덧셈
  } else if (typeof a === "string" && typeof b === "string") {
    return a + b; // 문자열 연결
  } else {
    return String(a) + String(b); // 혼합 타입 처리
  }
}
```

## 정리

if문, else if문 등 조건문으로 타입을 좁혀주는 것 = TypeGuard
유니온 타입처럼 할당 가능한 타입의 종류가 늘어나는 경우에 유용하게 사용
