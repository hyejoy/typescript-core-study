# TypeScript: 제네릭 (Generic)

## 제네릭이란?

**타입을 동적으로 정의**할 수 있게 해주는 기능.  
함수/클래스/인터페이스를 다양한 타입에 재사용할 수 있게 함.

---

## 제네릭 없을 때의 문제

```typescript
// 숫자 배열만 가능
function getFirst(arr: number[]): number {
  return arr[0];
}

// 문자열도 쓰려면 union 추가 → 코드가 길어짐
function getFirst(
  arr: number[] | string[] | boolean[],
): number | string | boolean {
  return arr[0];
}
```

---

## 제네릭 문법

```typescript
// 함수명 앞에 <T> 선언 → T가 동적 타입이 됨
function getFirst<T>(arr: T[]): T {
  return arr[0];
}
```

---

## 사용 예시

```typescript
// 호출 시 <타입>으로 T를 결정
getFirst<number>([1, 2, 3]); // T = number → (arr: number[]): number
getFirst<string>(["a", "b", "c"]); // T = string → (arr: string[]): string
getFirst<boolean>([true, false]); // T = boolean → (arr: boolean[]): boolean
```

### 타입 추론으로 생략도 가능

```typescript
getFirst([1, 2, 3]); // 전달값 보고 T = number 자동 추론
getFirst(["a", "b", "c"]); // T = string 자동 추론
```

> 명시적으로 `<number>` 처럼 적는 것을 권장 (타입이 명확하게 보임)

---

## 제네릭 타입 명칭

관례적으로 대문자를 사용. 여러 개일 때는 순서대로.

| 명칭 | 의미         |
| ---- | ------------ |
| `T`  | Type         |
| `K`  | Key          |
| `V`  | Value        |
| `U`  | 두 번째 타입 |

---

## 제네릭 vs union 타입 비교

```typescript
// union — 반환값 타입이 불명확
function getFirst(arr: number[] | string[]): number | string { ... }
const result = getFirst([1, 2, 3]); // → number | string (부정확)

// 제네릭 — 반환값 타입이 정확
function getFirst<T>(arr: T[]): T { ... }
const result = getFirst<number>([1, 2, 3]); // → number (정확)
```

> 제네릭은 타입의 **일관성**을 유지하면서도 **유연성**을 제공

---

## 함수 형태별 제네릭 작성법

핵심: **어떤 함수 형태든 소괄호 앞에 `<T>`** 를 붙임.

### 함수 선언문

```typescript
function identity<T>(value: T): T {
  return value;
}
```

### 함수 표현식

```typescript
const identity = function <T>(value: T): T {
  return value;
};
```

### 화살표 함수

```typescript
const identity = <T>(value: T): T => value;
```

### 변수에 타입 지정

```typescript
// 변수에 타입 지정 시 화살표 함수 형태로
const identity: <T>(value: T) => T = (value) => value;
```

---

## 제네릭 타입 추론

호출 시 `<타입>` 생략해도 자동 추론됨.

```typescript
identity<number>(1); // 명시적
identity(1); // 타입 추론으로 T = number 자동 결정
identity("a"); // T = string 자동 결정
identity(true); // T = boolean 자동 결정
identity([1, 2, 3]); // T = number[] 자동 결정
```

> 명시적 타입 지정과 타입 추론 모두 가능 — 개인 선호에 따라 선택
