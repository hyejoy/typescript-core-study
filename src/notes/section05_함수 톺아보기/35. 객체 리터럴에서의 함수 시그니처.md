# TypeScript: 객체 리터럴에서의 함수 시그니처

## 핵심 원칙

객체 속성의 함수를 어떤 방식으로 정의했든 **매개변수 + 반환값 타입 지정 방식은 동일**.

---

## 예시

```typescript
const calculator = {
  // 일반 함수
  add: function (n1: number, n2: number): number {
    return n1 + n2;
  },

  // 메서드 단축 문법
  subtract(n1: number, n2: number): number {
    return n1 - n2;
  },

  // 화살표 함수
  multiply: (n1: number, n2: number): number => {
    return n1 * n2;
  },
};
```

---

## 정리

| 함수 정의 방식    | 시그니처 작성 방식               |
| ----------------- | -------------------------------- |
| `function` 키워드 | 매개변수 타입 + `)` 뒤 반환 타입 |
| 메서드 단축 문법  | 동일                             |
| 화살표 함수       | 동일                             |

> 객체 안에서 함수를 어떻게 작성하든 시그니처 작성법은 **항상 같음**
