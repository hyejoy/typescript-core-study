# 클래스 (Class)

## 개념

반복된 속성의 객체를 만들 때 사용하는 ES6 문법
매번 중괄호로 객체를 정의하는 것보다 유지보수, 가독성 향상

## 기본 문법

```javascript
class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }
}

const per1 = new Person("철수", 20); // { name: '철수', age: 20 }
const per2 = new Person("영이", 30); // { name: '영이', age: 30 }
```

## 핵심 개념

- `class` 키워드 + 식별자로 정의
- `constructor` 메서드 → 객체 속성 초기값을 매개변수로 전달받음
- `new` 키워드로 호출 → 인스턴스 객체 생성
- 출력 시 앞에 붙는 `Person` → 해당 클래스에 의해 생성되었음을 나타내는 prefix
