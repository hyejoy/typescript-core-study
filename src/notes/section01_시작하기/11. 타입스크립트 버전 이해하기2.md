# TypeScript 버전 업데이트

## 설치된 패키지 최신 버전 확인

```bash
npm outdated
```

→ 현재 설치 버전 / 최신 버전 / 권장 버전 확인 가능

## 최신 버전으로 업데이트

```bash
npm install typescript@latest
```

→ uninstall 없이 바로 설치하면 자동으로 덮어씌워짐

## 버전 업데이트 전 vs 후

|                  | 5.5.4 (업데이트 전)     | 5.7.2 (업데이트 후) |
| ---------------- | ----------------------- | ------------------- |
| VSCode 에러 표시 | ✅ (내장 버전 5.8 기준) | ✅                  |
| 실행/컴파일 에러 | ❌ 없음                 | ✅ 발생             |

## 핵심 정리

에러 메시지 → VSCode 내장 TypeScript 버전 기준
실행/컴파일 → 프로젝트에 설치된 TypeScript 버전 기준
→ 두 버전 간 차이가 최소한으로 나도록 주기적으로 업데이트 필요
