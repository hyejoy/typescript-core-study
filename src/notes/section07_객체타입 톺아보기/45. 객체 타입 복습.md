# TypeScript: 객체 타입 지정 복습

## 기본 문법

```typescript
// 변수명: { 속성명: 타입 } = 값
const user: { name: string } = { name: "홍길동" };
```

---

## 속성 추가 — 세미콜론으로 구분

```typescript
const user: {
  name: string;
  age: number;
} = {
  name: "홍길동",
  age: 10,
};
```

---

## 함수 속성 추가

```typescript
const user: {
  name: string;
  age: number;
  getName: () => string; // 매개변수 없고 string 반환
} = {
  name: "홍길동",
  age: 10,
  getName() {
    return this.name;
  },
};
```

---

## 핵심 정리

| 속성 종류          | 타입 표기               |
| ------------------ | ----------------------- |
| 문자열             | `name: string`          |
| 숫자               | `age: number`           |
| 함수 (반환값 없음) | `fn: () => void`        |
| 함수 (반환값 있음) | `getName: () => string` |
| 객체 속성 구분자   | `;` (세미콜론)          |
