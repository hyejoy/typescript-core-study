# TypeScript: 객체 타입으로 함수 시그니처 작성

## 개념

JS에서 함수는 객체로 취급됨 → 객체 타입으로도 함수 시그니처 표현 가능.

```typescript
// 속성 명 쪽 = 매개변수 타입
// 속성 값 쪽 = 반환값 타입
변수명: { (매개변수: 타입): 반환타입 }
```

---

## 예시

```typescript
// 일반 함수
const add: { (n1: number, n2: number): number } = function (n1, n2) {
  return n1 + n2;
};

// 화살표 함수
const greet: { (name: string): string } = (name) => {
  return `Hello, ${name}!`;
};
```

---

## 일반 함수 시그니처 vs 객체 타입 함수 시그니처

```typescript
// 일반 방식 (권장)
const add: (n1: number, n2: number) => number = function (n1, n2) {
  return n1 + n2;
};

// 객체 타입 방식 (비일반적)
const add: { (n1: number, n2: number): number } = function (n1, n2) {
  return n1 + n2;
};
```

> 객체 타입으로도 가능하지만 **일반적이지 않음**  
> 앞에서 배운 방식으로 작성하는 것을 권장  
> **"이런 방법도 있다"** 정도로만 기억
