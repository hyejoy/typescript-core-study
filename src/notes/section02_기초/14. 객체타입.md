# TypeScript: 객체(Object) 타입

## 기본 문법

객체의 타입은 **속성마다 타입을 지정**하는 형태.

```typescript
// 객체 값
const person = { name: "홍길동" };

// 객체 타입 지정
const person: { name: string } = { name: "홍길동" };
```

> 객체 값과 비슷하게 생겼지만, 값 자리에 **타입**이 들어가는 것이 핵심

---

## 객체 vs 객체 타입 구분자

|             | 객체 (값)                     | 객체 타입                       |
| ----------- | ----------------------------- | ------------------------------- |
| 속성 구분자 | `,` (콤마)                    | `;` (세미콜론)                  |
| 예시        | `{ name: "홍길동", age: 20 }` | `{ name: string; age: number }` |

---

## 예시 모음

### 빈 객체

```typescript
const obj: {} = {};
```

### 단일 속성

```typescript
const person: { name: string } = { name: "홍길동" };
```

### 여러 속성

```typescript
const person: {
  name: string;
  age: number;
  adult: boolean;
} = {
  name: "홍길동",
  age: 20,
  adult: true,
};
```

### undefined / null 속성

```typescript
const person: {
  name: string;
  job: undefined; // undefined 타입
} = {
  name: "홍길동",
  job: undefined,
};

const person: {
  name: string;
  job: null; // null 타입
} = {
  name: "홍길동",
  job: null,
};
```

### 중첩 객체 (객체 안에 객체)

```typescript
const person: {
  name: string;
  age: number;
  adult: boolean;
  job: {
    // 중첩 객체도 동일하게 타입 지정
    title: string;
    vue: number;
  };
} = {
  name: "홍길동",
  age: 20,
  adult: true,
  job: {
    title: "개발자",
    vue: 5000,
  },
};
```

### Date 타입

```typescript
const person: {
  name: string;
  today: Date; // 표준 내장 객체 Date 타입
} = {
  name: "홍길동",
  today: new Date(),
};
```

---

## 정리

| 값           | 타입                  |
| ------------ | --------------------- |
| `"홍길동"`   | `string`              |
| `20`         | `number`              |
| `true`       | `boolean`             |
| `undefined`  | `undefined`           |
| `null`       | `null`                |
| `new Date()` | `Date`                |
| `{ ... }`    | `{ 속성: 타입; ... }` |
