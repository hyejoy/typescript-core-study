# TypeScript: 배열(Array) 타입 지정

## 두 가지 방법

| 방법                 | 문법          | 예시            |
| -------------------- | ------------- | --------------- |
| **기본 배열 타입**   | `타입[]`      | `number[]`      |
| **제네릭 배열 타입** | `Array<타입>` | `Array<number>` |

> 일반적으로 **기본 배열 타입** 방식을 더 선호

---

## 요소 타입별 예시

```typescript
// 숫자 배열
const nums: number[] = [1, 2, 3];
const nums: Array<number> = [1, 2, 3];

// 문자 배열
const strs: string[] = ["a", "b"];
const strs: Array<string> = ["a", "b"];

// 논리 배열
const bools: boolean[] = [true, false];

// null 배열
const nulls: null[] = [null, null];

// 객체 배열
const users: { name: string }[] = [{ name: "홍길동" }];
const users: Array<{ name: string }> = [{ name: "홍길동" }];

// 함수 배열 (반환값 없는 함수)
const fns: (() => void)[] = [() => console.log("hi")];
const fns: Array<() => void> = [() => console.log("hi")];
```

---

## 여러 타입이 섞인 배열 → Tuple

배열 인덱스마다 타입을 직접 지정하는 방식.

```typescript
// [number, string, boolean] 순서로 고정
const mixed: [number, string, boolean] = [1, "hello", true];
//            0번     1번      2번
```

### Tuple 특징

- 각 인덱스의 **타입과 순서가 고정**됨
- **기본 배열 타입 방식으로만** 가능 (`Array<>` 방식 불가)
- 잘못된 순서나 타입으로 할당하면 에러 발생

```typescript
const mixed: [number, string, boolean] = ["hello", 1, true]; // ❌ 에러
```

---

## 정리

```
단일 타입 배열   → 타입[]          예) number[], string[]
단일 타입 배열   → Array<타입>     예) Array<number>  (덜 선호)
혼합 타입 배열   → [타입, 타입, ...] (Tuple)
```
