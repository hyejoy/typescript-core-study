# TypeScript: `never` 타입

## never 타입이란?

**절대로 발생하지 않는 값 / 절대로 불가능한 경우**를 나타내는 타입.

```typescript
const a: never = 30; // ❌ 에러
const a: never = "hello"; // ❌ 에러
const a: never = true; // ❌ 에러
const a: never = []; // ❌ 에러
const a: never = {}; // ❌ 에러

// any, unknown에도 할당 불가
const b: any = a; // ❌ 에러
const c: unknown = a; // ❌ 에러
```

→ never로 지정된 변수에는 **어떤 값도 할당 불가**, **다른 변수에 할당도 불가**

---

## never를 사용하는 경우

### 1. 예외를 던지는 함수 (실무에서 주로 사용)

```typescript
// void ❌ — 단순히 반환값이 없는 경우
// never ✅ — 예외를 던져서 정상 반환 자체가 불가능한 경우
function throwError(message: string): never {
  throw new Error(message);
}
```

### 2. 무한 루프 함수

```typescript
function infiniteLoop(): never {
  while (true) {
    // 영원히 끝나지 않음 → 반환 자체가 불가능
  }
}
```

---

## void vs never

|           | `void`                    | `never`                     |
| --------- | ------------------------- | --------------------------- |
| 의미      | 반환값이 없음             | 반환 자체가 불가능          |
| 사용 경우 | 일반적인 반환값 없는 함수 | 예외 던지는 함수, 무한 루프 |
| 예시      | `console.log(...)`        | `throw new Error(...)`      |

---

## 지금 당장 기억할 것

> **함수 내부에서 예외를 던지는 경우 → 반환 타입을 `never`로 지정**
