# TypeScript: `intersection` 타입

## intersection 타입이란?

`&` (ampersand) 기호를 사용하여 **여러 타입을 모두 만족**해야 하는 타입.  
JS의 `&&` AND 연산자와 비슷한 개념.

```typescript
// name 속성만 있는 타입
const a: { name: string } = { name: "홍길동" }; // ✅

// name AND age 둘 다 있어야 하는 타입
const a: { name: string } & { age: number } = {
  name: "홍길동",
  age: 20,
}; // ✅

const a: { name: string } & { age: number } = {
  name: "홍길동",
  // age 없음 → ❌ 에러 (age 프로퍼티 누락)
};
```

---

## union vs intersection 비교

|                | union (`\|`)            | intersection (`&`)               |
| -------------- | ----------------------- | -------------------------------- |
| 기호           | `\|`                    | `&`                              |
| JS 연산자 비유 | `\|\|` (OR)             | `&&` (AND)                       |
| 의미           | 나열된 타입 **중 하나** | 나열된 타입 **모두 만족**        |
| 예시           | `number \| string`      | `{name: string} & {age: number}` |

---

## 여러 타입 연결

```typescript
const a: { name: string } & { age: number } & { gender?: string } = {
  name: "홍길동",
  age: 20,
  // gender는 옵셔널이므로 없어도 OK
};

// 최종 타입 = name + age + gender? 모두 포함
```

---

## 정리

```
union        → A 또는 B      (하나만 만족해도 됨)
intersection → A 그리고 B    (둘 다 만족해야 함)
```
