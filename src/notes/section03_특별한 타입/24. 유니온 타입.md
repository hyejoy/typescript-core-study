# TypeScript: `union` 타입

## union 타입이란?

하나의 변수가 **여러 타입 중 하나**를 가질 수 있도록 허용하는 타입.  
`|` (pipe) 기호로 표기.

```typescript
// number만 허용
const a: number = 10;
a = "hello"; // ❌ 에러

// number 또는 string 허용
const a: number | string = 10;
a = "hello"; // ✅ OK
```

---

## 여러 타입 연속 사용

```typescript
// number 또는 string 또는 boolean
const a: number | string | boolean = 10;
a = "hello"; // ✅
a = true; // ✅
```

---

## 다양한 활용 예시

```typescript
// 함수 매개변수
function printValue(value: number | string): void {
  console.log(value);
}

// 배열 요소
const arr: (number | string)[] = [1, "hello", 2, "world"];

// 객체 속성
const user: {
  name: string;
  age: number | null; // 나이가 없을 수도 있는 경우
} = {
  name: "홍길동",
  age: null,
};
```

---

## JS의 OR 연산자와 비슷한 개념

```
number | string | boolean
→ number 또는 string 또는 boolean
→ JS의 || (OR 연산자) 와 같은 원리
```

---

## 정리

|      | 설명                                         |
| ---- | -------------------------------------------- |
| 문법 | `타입1 \| 타입2 \| 타입3 ...`                |
| 의미 | 나열된 타입 중 하나의 값 허용                |
| 장점 | 타입의 유연성을 높여 효율적인 코드 작성 가능 |
