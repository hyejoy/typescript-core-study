# TypeScript: 객체 속성의 함수 · 배열 타입 지정

## 객체 속성에 함수가 있을 때

```typescript
const obj: {
  job: string;
  getJob: () => string; // 매개변수 없고 string 반환하는 함수
} = {
  job: "개발자",
  getJob() {
    return this.job;
  },
};
```

- 함수 속성의 타입은 **화살표 함수 형태**로 지정
- `() => 반환타입` 형식

---

## 객체 속성에 배열이 있을 때

```typescript
const obj: {
  job: string;
  getJob: () => string;
  skills: string[]; // 문자열 배열
} = {
  job: "개발자",
  getJob() {
    return this.job;
  },
  skills: ["React", "TypeScript"],
};
```

---

## 핵심 원칙

> 객체 속성의 타입 지정은 **각 속성의 값 타입을 그대로 지정**하면 됨.  
> 함수든 배열이든 앞서 배운 방식 그대로 적용.

```
속성값이 string   → string
속성값이 number   → number
속성값이 함수     → (매개변수: 타입) => 반환타입
속성값이 배열     → 타입[]
속성값이 객체     → { 속성: 타입 }
```

---

## 종합 예시

```typescript
const developer: {
  name: string;
  age: number;
  skills: string[]; // 배열
  getJob: () => string; // 반환값 있는 함수
  printInfo: () => void; // 반환값 없는 함수
  job: {
    // 중첩 객체
    title: string;
    salary: number;
  };
} = {
  name: "홍길동",
  age: 30,
  skills: ["React", "TypeScript"],
  getJob() {
    return "개발자";
  },
  printInfo() {
    console.log(this.name);
  },
  job: { title: "프론트엔드", salary: 5000 },
};
```
