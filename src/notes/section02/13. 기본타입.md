# TypeScript: 기본 자료형 타입

## JS 값의 종류

```
기본 자료형 (Primitive)   → 숫자, 문자, 논리, Undefined, Null, Symbol
참조 자료형 (Reference)   → 배열, 객체, 함수  (다음 강의에서 다룸)
```

---

## 타입 지정 문법

```typescript
// 변수명: 타입 = 값
const 변수명: 타입 = 값;
```

---

## 기본 자료형 타입 목록

| 값 종류   | TypeScript 타입 | 예시                                |
| --------- | --------------- | ----------------------------------- |
| 숫자      | `number`        | `const num: number = 30;`           |
| 문자열    | `string`        | `const str: string = "hello";`      |
| 논리      | `boolean`       | `const bool: boolean = true;`       |
| undefined | `undefined`     | `const und: undefined = undefined;` |
| null      | `null`          | `const n: null = null;`             |
| symbol    | `symbol`        | `const sym: symbol = Symbol();`     |

> JS의 `typeof` 키워드로 출력한 타입과 거의 동일

---

## 타입 지정의 효과

타입이 지정된 변수에는 **해당 타입의 값만 할당 가능**.

```typescript
const num: number = 30;
num = "hello"; // ❌ 에러 — number 타입에 string 할당 불가
num = 50; // ✅ OK
```

### 핵심 규칙

- 한번 지정된 타입은 **변경 불가**
- 새 변수를 선언하기 전까지 최초 타입 유지

---

## TypeScript의 역할

```
JavaScript: 타입 신경 안 씀 → 런타임에 오류 발생 가능
TypeScript: 타입 명시 → 코드 작성 중(컴파일 타임)에 오류 발견
```
