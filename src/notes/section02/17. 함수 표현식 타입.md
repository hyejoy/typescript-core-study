# TypeScript: 함수 표현식 타입 지정

## 함수 선언문과 동일한 부분

매개변수와 반환값에 타입을 지정하는 방식은 동일.

```typescript
// 함수 표현식 — 함수에 직접 타입 지정
const getSum = function (a: number, b: number): number {
  return a + b;
};

const printSum = function (a: number, b: number): void {
  console.log(a + b);
};
```

---

## 함수 표현식만의 특징 — 변수에도 타입 지정 가능

변수에 타입을 지정할 때는 **화살표 함수 형태**로 작성.

```typescript
// 변수에 타입 지정하는 문법
const 변수명: (매개변수: 타입) => 반환값타입 = function (...) { ... };
```

```typescript
// 예시
const getSum: (a: number, b: number) => number = function (a, b) {
  return a + b;
};

// 매개변수 없고 반환값도 없을 때
const printHello: () => void = function () {
  console.log("hello");
};
```

---

## 타입 지정 위치 — 둘 중 하나만 선택

| 방식        | 코드                                              |
| ----------- | ------------------------------------------------- |
| 함수에 직접 | `const fn = function(a: number): void { }`        |
| 변수에 지정 | `const fn: (a: number) => void = function(a) { }` |

> ⚠️ 둘 다 동시에 작성하는 것은 불필요한 중복 → **권장하지 않음**

---

## 화살표 함수도 동일

```typescript
// 함수에 직접
const getSum = (a: number, b: number): number => a + b;

// 변수에 지정
const getSum: (a: number, b: number) => number = (a, b) => a + b;
```

화살표 함수로 변경해도 타입 지정 방식은 동일.

---

## 함수 선언문 vs 함수 표현식 타입 지정 비교

|                  | 함수 선언문 | 함수 표현식 |
| ---------------- | ----------- | ----------- |
| 매개변수 타입    | ✅          | ✅          |
| 반환값 타입      | ✅          | ✅          |
| 변수에 타입 지정 | ❌          | ✅          |
