# TypeScript: 함수 선언문 타입 지정

## 함수 타입 지정 위치

함수 선언문에서는 **매개변수**와 **반환값** 두 곳에 타입을 지정.

```typescript
function 함수명(매개변수: 타입): 반환값타입 {
  // ...
}
```

---

## 반환값 없을 때 → `void`

```typescript
function printSum(a: number, b: number): void {
  console.log(a + b); // 출력만 하고 반환값 없음
}
```

- `void` = 반환값이 없다는 의미의 타입

---

## 반환값 있을 때

```typescript
function getSum(a: number, b: number): number {
  return a + b; // number 타입 반환
}
```

- 반환되는 값의 타입을 미리 예측해서 지정

---

## 타입 지정 정리

| 위치     | 문법             | 예시                   |
| -------- | ---------------- | ---------------------- |
| 매개변수 | `매개변수: 타입` | `a: number`            |
| 반환값   | `): 타입 {`      | `): void`, `): number` |

---

## 반환값 타입 종류

| 상황        | 타입             |
| ----------- | ---------------- |
| 반환값 없음 | `void`           |
| 숫자 반환   | `number`         |
| 문자 반환   | `string`         |
| 논리 반환   | `boolean`        |
| 객체 반환   | `{ 속성: 타입 }` |
