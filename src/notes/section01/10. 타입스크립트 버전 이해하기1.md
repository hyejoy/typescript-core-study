# TypeScript 버전 이해하기

## 두 가지 TypeScript 버전

### 1. 프로젝트에 설치된 버전

- `package.json`에 설치된 TypeScript 버전
- **코드 실행/컴파일 시** 이 버전에 의존

### 2. VSCode 내장 버전

- VSCode 자체에 내장된 TypeScript 버전
- **에러 메시지 표시 시** 이 버전에 의존
- VSCode 업데이트와 연동 → 오랫동안 업데이트 안 했으면 낮은 버전

## VSCode 내장 TypeScript 버전 확인 방법

톱니바퀴 아이콘 → 커맨드 팔레트 → `Select TypeScript Version` 검색 → 현재 버전 확인

## 핵심 정리

에러 메시지 → VSCode 내장 TypeScript 버전 기준
실제 실행/컴파일 → 프로젝트에 설치된 TypeScript 버전 기준

## 예시

- VSCode 내장 버전: 5.8 (5.7에서 추가된 기능 → 에러 표시)
- 프로젝트 설치 버전: 5.5.4 (해당 기능 없음 → 실행 시 에러 없음)
- 결과: VSCode에서는 에러 표시되지만 실행하면 정상 동작

## 주의사항

두 버전 간 차이가 많이 나지 않도록 주기적으로 프로젝트 TypeScript 버전 관리 필요
