````markdown
# Code Runner로 TypeScript 실행하기

## 필요한 것

- VSCode **Code Runner** 익스텐션 설치
- **Node.js** 설치 확인

```bash
node -v  # 버전 출력되면 OK
```
````

- **ts-node** 패키지 설치

```bash
npm install ts-node --save-dev
```

## settings.json 설정

VSCode 설정 → 우측 상단 버튼 클릭 → settings.json 열기

```json
{
  "code-runner.clearPreviousOutput": true,
  "code-runner.executorMap": {
    "typescript": "./node_modules/.bin/ts-node"
  }
}
```

## 실행 방법

실행할 `.ts` 파일 열기 → 우측 상단 **▶ 버튼** 클릭

## ⚠️ 주의사항

node_modules 폴더는 반드시 프로젝트 루트 경로에 있어야 함
→ 다른 폴더 안에 있으면 실행 안 됨

## 컴파일 방식 vs Code Runner 방식

|          | 컴파일 방식                | Code Runner      |
| -------- | -------------------------- | ---------------- |
| 방법     | `npx tsc` → dist 폴더 생성 | ▶ 버튼 클릭      |
| 용도     | 실제 배포                  | 빠른 실행/테스트 |
| 번거로움 | 매번 컴파일 필요           | 바로 실행 가능   |
