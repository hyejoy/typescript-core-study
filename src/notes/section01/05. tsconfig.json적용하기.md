# tsconfig.json 설정하기

## tsconfig.json 생성

```bash
npx tsc --init
```

생성 후 파일 안에 많은 컴파일 옵션이 자동으로 작성됨
→ 우측 주석으로 각 옵션 설명 확인 가능

---

## 주요 옵션 설정

```json
{
  "compilerOptions": {
    "target": "ESNext", // 최신 ECMAScript 표준으로 컴파일
    "outDir": "dist" // 컴파일 결과를 dist 폴더에 생성
  },
  "include": ["src/**/*"], // src 폴더 하위 모든 ts 파일 컴파일 대상
  "exclude": ["node_modules"] // node_modules는 컴파일 제외
}
```

---

## 컴파일 방법

```bash
npx tsc
```

- `src` 폴더 하위 `.ts` 파일 전체 컴파일
- 컴파일 결과는 `dist` 폴더에 생성
- `target: ESNext` 설정으로 최신 문법으로 컴파일

---

## tsconfig.json 적용 전 vs 후

적용 전
→ var 키워드 (ES5 이전 문법)로 컴파일
적용 후
→ ESNext 최신 표준 문법으로 컴파일

---

## 실행 흐름

.ts 파일 작성 (src/)
↓
npx tsc (컴파일)
↓
.js 파일 생성 (dist/)
↓
index.html에서 dist/ 경로로 참조하여 실행
