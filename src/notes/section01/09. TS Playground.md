# TypeScript Playground

## 개념

브라우저에서 TypeScript 코드를 즉시 작성하고 실행할 수 있는 웹 기반 개발 도구
별도 설치 없이 웹페이지 접속만으로 사용 가능

## 접속 방법

구글에 TypeScript 검색 → 공식 홈페이지 → 상단 Playground 메뉴 클릭

## 주요 탭 설명

- **JS** : 작성한 TypeScript 코드가 JavaScript로 변환된 결과 실시간 확인
- **DTS** : 외부 라이브러리/모듈의 타입 정보 (활용도 낮음)
- **에러** : 오류 발생 시 오류 개수 및 상세 내용 표시
- **Logs** : Run 버튼 클릭 시 console.log 출력 결과 표시
- **플러그인** : 플레이그라운드에서 사용할 플러그인 설정/해제

## TSConfig 설정

TSConfig 메뉴에서 tsconfig.json과 동일한 설정 가능
→ 보통은 기본값으로 사용해도 무방

## 장점

- 설치 없이 바로 사용 가능
- 버전별 TypeScript 환경 빠르게 변경 가능
- 실시간으로 JS 변환 결과 확인
- 오류 즉시 확인 가능
