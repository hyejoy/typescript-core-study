## 각 타입별 추론 결과

| 값               | 추론되는 타입      |
| ---------------- | ------------------ |
| `10`             | `number`           |
| `"hello"`        | `string`           |
| `true`           | `boolean`          |
| `undefined`      | `any` ⚠️           |
| `null`           | `any` ⚠️           |
| `Symbol()`       | `symbol`           |
| `[1, 2, 3]`      | `number[]`         |
| `{ name: "홍" }` | `{ name: string }` |
| `() => {}`       | `() => void`       |
