# TypeScript: 타입 추론 (Type Inference)

## 타입 추론이란?

개발자가 타입을 명시하지 않아도 **TypeScript 컴파일러가 할당된 값을 보고 자동으로 타입을 추론**해주는 기능.

```typescript
// 타입 명시 O
const num: number = 10;
const str: string = "hello";

// 타입 명시 X → 컴파일러가 자동 추론
const num = 10; // → number로 추론
const str = "hello"; // → string으로 추론
```

---

## 추론 예시

```typescript
const num = 10; // number
const str = "hello"; // string
const bool = true; // boolean
const und = undefined; // undefined
const nul = null; // null
const sym = Symbol(); // symbol

const arr = [1, 2, 3]; // number[]
const obj = { name: "홍길동" }; // { name: string }
const fn = () => "hello"; // () => string
```

## VSCode에서 추론 타입 확인하는 법

변수에 마우스를 올리면 추론된 타입이 툴팁으로 표시됨.

> VSCode가 내부적으로 TypeScript 컴파일러를 사용하기 때문에 가능

## 타입 추론 vs 명시적 타입 지정

|             | 타입 추론             | 명시적 타입 지정 |
| ----------- | --------------------- | ---------------- |
| 코드 길이   | 짧음                  | 김               |
| 타입 안전성 | ✅ (추론이 정확할 때) | ✅               |
| 가독성      | 상황에 따라 다름      | 명확함           |

> 타입 추론이 정확하게 동작할 때는 굳이 타입을 명시하지 않아도 됨  
> 단, 복잡한 타입이나 의도를 명확히 해야 할 때는 명시적으로 지정하는 것이 좋음
