# TypeScript: `readonly` 키워드

## readonly란?

객체 타입의 속성에 `readonly`를 붙이면 **해당 속성값을 수정 불가능**하게 만드는 키워드.

---

## 사용법

```typescript
const user: {
  name: string;
  readonly age: number; // age는 읽기 전용
} = {
  name: "홍길동",
  age: 20,
};

user.name = "철수"; // ✅ name은 수정 가능
user.age = 30; // ❌ 에러 — age는 readonly 프로퍼티라 할당 불가
```

---

## 인터페이스에서도 사용 가능

```typescript
interface User {
  name: string;
  age: number;
  readonly gender: male | female; // 읽기 전용
}

const user: User = { name: "홍길동", age: 20, gender: "male" };

user.name = "이철수"; // ✅ 수정 가능
user.gender = "female"; // ❌ 에러 — readonly property
```

---

## 정리

|      | 일반 속성     | `readonly` 속성       |
| ---- | ------------- | --------------------- |
| 읽기 | ✅            | ✅                    |
| 수정 | ✅            | ❌                    |
| 용도 | 일반적인 속성 | 변경되면 안 되는 속성 |

> `readonly`는 인터페이스뿐만 아니라 **객체 타입 어디서든 사용 가능**
